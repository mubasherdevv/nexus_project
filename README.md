# Nexus - Connecting Entrepreneurs & Investors

Nexus is a premium platform designed to bridge the gap between visionary entrepreneurs and strategic investors. It provides a seamless environment for pitching ideas, managing investments, and communicating in real-time.

## 🚀 Key Features

- **Real-Time Communication**: WhatsApp-style chat system with message status (Sent, Delivered, Read), typing indicators, and user presence (Online/Last Seen).
- **Investor Dashboard**: Comprehensive view of portfolio size, active startups, and upcoming deal flows.
- **Financial Suite**: Robust wallet system for deposits, withdrawals, and secure transfers between users.
- **Meeting Management**: Integrated scheduling for pitch meetings with real-time notifications and status tracking.
- **Dynamic Notifications**: Instant alerts for all platform activities, ensuring you never miss an opportunity.
- **Premium UI**: Modern, glassmorphism-inspired design with a focus on usability and aesthetics.

## 🛠️ Technology Stack

- **Frontend**: React, Vite, Tailwind CSS, Lucide Icons.
- **Backend**: Node.js, Express, MongoDB.
- **Real-Time**: Socket.io for instant messaging and notifications.
- **Security**: JWT Authentication and secure transaction logs.

## 📦 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB (Local or Atlas)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mubasherdevv/nexus_project.git
   ```
2. Install dependencies for both frontend and backend:
   ```bash
   npm install
   cd server && npm install
   ```
3. Set up your `.env` files in the root and server directories.
4. Run the development servers:
   ```bash
   # Root directory (Frontend)
   npm run dev
   
   # Server directory (Backend)
   cd server && npm run dev
   ```
